Cascading style sheets atau CSS adalah bahasa pemograman berbasis aturan yang dikembangkan oleh World Wide Web (W3C) pada tahun 1996. CSS dikembangkan karena HTML yang berfungsi untuk menformat halaman web tidak bisa menambahkan gaya atau tata letak tingkat lanjut. Misalnya, pada HTML versi 3.2, sudah diperkenalkan beberapa elemen presentasi seperti font, center, dan marquee yang memungkinkan pengembang web untuk menambahkan gaya pada halaman mereka. Namun, penggunaan elemen-elemen ini tidak dianjurkan karena dapat menyebabkan masalah aksesibilitas dan kesulitan dalam pemeliharaan kode.
